# Simulador de Passageiros Titanic

Um aplicativo interativo com interface gráfica construído em Python. Este projeto permite que os usuários simulem passageiros a bordo do RMS Titanic e utilizem modelos de Machine Learning (Regressão Logística e Árvores de Decisão) com diferentes parâmetros para prever se o passageiro sobreviveria ou não ao naufrágio.


## Pré-requisitos

- Python 3.8+
- pip (gerenciador de pacotes Python)


## Instalação

### 1. Clone ou baixe o repositório

```bash
cd clube-ml-titanic
```

### 2. (Opcional) Criar e ativar um ambiente virtual

**Windows (PowerShell):**
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

**Windows (Prompt de Comando):**
```cmd
python -m venv .venv
.\.venv\Scripts\activate.bat
```

**macOS / Linux:**
```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Instalar dependências

```bash
pip install -r requirements.txt
```


## Como Executar

Execute a interface gráfica do projeto:

```bash
python gui.py
```

A aplicação abrirá uma janela com a interface para simular passageiros e fazer previsões.

## Ao finalizar a execução desative o ambiente virtual (venv)

```bash
deactivate
```
